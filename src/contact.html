Contact Us


Have a general question or need help?

If you require assistance in determining your Security needs or have a general inquiry please send an email to security-help[@]wikimedia.org.

Interested in detailed information on requests and workflows?

Our Request for Service SOP explains how to request work from the Security Team.

Want to report a security issue?

For more information on how to report a security issue in MediaWiki or Wikimedia sites, please see the Reporting Security Bugs page on MediaWiki.org or at HackerOne.
